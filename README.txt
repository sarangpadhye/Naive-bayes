Naive Bayes      |
----------------------------------------------|

The purpose of this assignment is to build a Naive Bayes Classifier in order to classify a two class label.

Procedure to run the code:
-----------------------------------------
The submission consists of 2 java files.
1) DataRep.java
2) NaiveBayes.java
The main procedure is contained in NaiveBayes.java so this file should be run through any Java IDE such as eclipse.

File path variable names :
-----------------------------------------
1) The training and testing set is declared in the variable "datapath" in the method LoadData(). Both training set and test Set are loaded
in the same method. Training set can be loaded by passing the 0 as the parameter to the method while test data can be loaded by passing
the parameter 1 to the method.
